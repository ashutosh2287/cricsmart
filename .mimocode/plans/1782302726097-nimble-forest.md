# Plan: Cinematic UI Redesign — All Pages

## Goal
Transform CricSmart from a plain card-based layout into a professional ESPNcricinfo + broadcast hybrid with cinematic dark theme, dramatic animations, and sports-grade visual hierarchy.

## Design System
- **Theme**: Dark cinematic (#040A14 base, #0A1628 surface)
- **Accents**: Cyan (#00E5FF), Purple (#7C3AED), Green (#00FF87), Amber (#F59E0B), Red (#EF4444)
- **Fonts**: Space Grotesk (headings), DM Sans (body)
- **Animations**: framer-motion (already installed, 37+ files use it)
- **Motion library**: `src/components/ui/motion.ts` (258 lines of variants)

---

## Phase 1: Unify Color System + Global Styles

### 1.1 Rewrite `src/app/globals.css`
- Replace the green-based light/dark tokens with the dark cinematic palette
- Keep light mode tokens but make them complementary
- Add new cinematic utility classes: `.glass`, `.glow-border`, `.neon-text`, `.card-cinematic`
- Add CSS keyframe animations: `shimmer`, `pulse-glow`, `float`, `slide-up-reveal`
- Add gradient utilities: `.gradient-brand`, `.gradient-score`, `.gradient-hero`

### 1.2 Update `src/components/ui/motion.ts`
- Add new cinematic variants: `heroReveal`, `scoreFlash`, `cardFlip`, `staggerIn`, `parallaxFade`
- Add `cinematicTransition` preset for dramatic page transitions

**Files to modify:**
