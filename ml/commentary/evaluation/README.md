# Commentary Evaluation

Quality, tone, continuity and fallback reliability reports.

Tracked gates:
- repetition rate
- contextual accuracy
- tone accuracy
- narrative alignment
- commentary diversity
- retrieval relevance

Utilities:
- `dataset_validation.py` (fails training on quality issues)
- `replay_validation.py` (deterministic replay parity)
- `generate_eval_report.py` (offline JSON + markdown report)
